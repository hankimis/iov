# IOV - Multi Channel Network

IOV는 아이디어를 입력(Input)하고, 기술로 처리하여(Output), 가치 있는 결과(Value)로 전환하는 종합 미디어 컨텐츠 네트워크(MCN)입니다.

## 🚀 주요 기능

### 웹사이트 페이지
- **HOME**: 회사 소개 및 핵심 서비스 개요
- **ABOUT**: 회사 소개, 미션, 비전, 핵심 가치
- **BUSINESS**: 3가지 핵심 사업 영역 상세 소개
  - 엔터테인먼트 (인플루언서 육성 및 매니지먼트)
  - 마케팅 대행업 (컨텐츠 마케팅)
  - 영상 제작사 (기획부터 편집까지)
- **CREATORS**: 소속 크리에이터 소개 및 통계
- **NEWS**: 회사 소식 및 업데이트
- **RECRUIT**: 크리에이터 지원 및 비즈니스 파트너 신청
- **CONTACT**: 연락처 정보 및 문의 양식

## 🛠️ 기술 스택

- **Framework**: Next.js 16 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS 4
- **아이콘**: Lucide React
- **애니메이션**: Framer Motion

## 📦 설치 및 실행

### 개발 환경 실행
```bash
npm install
npm run dev
```

개발 서버가 [http://localhost:3000](http://localhost:3000)에서 실행됩니다.

### 프로덕션 빌드
```bash
npm run build
npm start
```

### 린트 검사
```bash
npm run lint
```

## 🎨 디자인 특징

## 🌟 주요 사업 영역

### 1. 엔터테인먼트
- 틱톡/유튜브 라이브 운영
- 인플루언서 육성 및 매니지먼트
- 광고 수주 및 기획 대행

### 2. 마케팅 대행업
- 요식업/의류/개인사업자 대상 컨텐츠 마케팅
- 인플루언서 공구 및 협찬
- 바이럴 컨텐츠 기획 및 편집

### 3. 영상 제작사
- 외부 영상 제작 (기획, 편집)
- 지역방송국 방송 편집 대행
- 콘서트/행사/결혼식 등 외부 촬영
